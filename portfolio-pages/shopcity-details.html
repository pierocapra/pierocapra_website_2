Shopcity E-Commerce

Project information

  • Category: Full Stack
  • Project date: 18 Oct, 2021
  • Project URL: shopcity-ecommerce.herokuapp.com

Shopcity E-Commerce is a web application which will simulate a fully functional e-commerce app. The project has been realized using Angular for the Front-End and for the Back-End Java Spring Boot and MySql Database.
